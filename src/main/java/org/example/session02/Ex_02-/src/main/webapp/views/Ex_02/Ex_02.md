| # | Vị trí                                   | Loại vấn đề         | Mô tả hậu quả                                                                      |
| - | ---------------------------------------- | ------------------- | ---------------------------------------------------------------------------------- |
| 1 | `<%! private int requestCounter = 0; %>` | Race Condition      | Biến dùng chung cho nhiều request, dễ sai dữ liệu khi nhiều user truy cập cùng lúc |
| 2 | `<% ... %>` (scriptlet)                  | Logic trong View    | Nhét Java vào JSP, code khó đọc, khó bảo trì                                       |
| 3 | `<%= ... %>`                             | Expression lỗi thời | Không dùng EL, khó kiểm soát, khó bảo mật                                          |
| 4 | `<!-- comment -->`                       | Sai loại comment    | HTML comment vẫn hiển thị trong source, lộ logic                                   |
| 5 | In dữ liệu trực tiếp                     | XSS                 | Nếu dữ liệu từ user, có thể bị chèn script độc                                     |
| 6 | Vòng lặp Java trong JSP                  | Sai kiến trúc       | JSP không nên xử lý logic                                                          |
